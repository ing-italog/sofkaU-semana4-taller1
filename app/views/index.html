GUIDE API - DICES GAME


Guiden on how to use the api.
ROUTES

La ruta raiz es: http://localhost:defaultPort/api

Peticiones GET

  • / esta url mostrara el historial de partidas

  • / +id esta ruta devuelve una pártida en especifico, /id en la cabecera debes incluir el id que se desea consultar

  • / +id /winner se utiliza para definir el ganador de una pártida, el ganador se da de forma aleatoria, por lo que en la petición unicámente se debera poner el /id de la partida, más /winner

Peticiones POST

  • /createGame url para crear una pártida nueva, puedes usar la siguiente ilustración como guía para hacer la envíar los datos json
    { " gamers ":[ {" name ": " example1 ", " bet " : 12345} ]

  • /startGame se debe enviar el id en el body de la petición, el ganador se se define de forma aleatoria.
    { " id ":{" id example "} }
